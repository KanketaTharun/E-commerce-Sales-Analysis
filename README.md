# 📊 E-Commerce Analytics Dashboard

## 📌 Project Overview

This is an end-to-end E-Commerce Analytics project developed using PostgreSQL and Power BI. The project focuses on transforming raw e-commerce transaction data into actionable business insights through interactive dashboards and KPI reporting.

The dashboards help stakeholders monitor revenue, profit, orders, customer behavior, and overall sales performance.

---

## 🛠️ Tools & Technologies Used

- PostgreSQL
- Power BI
- DAX
- SQL
- Data Modeling
- Data Visualization

---

## 📂 Dataset

The dataset was generated and managed in PostgreSQL and includes:

- Customers
- Orders
- Order Details
- Products
- Categories
- Date Table
